# NovelVerse
NovelVerse is a simple and interactive website made for book lovers who enjoy discovering novels from different genres.
The website focuses on making it easier for users to search books, explore authors and enjoy a smooth reading experience through a clean and modern interface.

## Features
Responsive website design
Search functionality
Genre filtering
Book cards with description
Author profiles
Contact form validation
Smooth scrolling navigation

## Technologies used
HTML5
CSS
JavaScript
